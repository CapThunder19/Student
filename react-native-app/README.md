# Student React Native App

This folder contains a React Native demo version of the Student website UI.

Included mobile modules:

- Auth with demo login and signup
- Home with website-style cards
- Community with local chat, replies, and likes
- Budget with local target and expense tracking
- SOS with local emergency reports
- Profile with local editing and persistence

## Setup

1. `cd react-native-app`
2. `npm install`
3. `npm run start`
